# 커밋 컨벤션

- 기반: Conventional Commits (`type(scope): summary`)
- 허용 type: `feat`, `fix`, `docs`, `style`, `refactor`, `perf`, `test`, `chore`

## 스코프 예시

- `fsd`, `react-query`, `templates`, `docs`, `tasks`, `ui`

## 본문/푸터

- 이슈/PRD 링크: `Refs: #<issue>` or `Refs: docs/tasks/NNN-slug.md`
- 한 커밋 = 한 논리 단위 (문서/설치/코드 분리)

## 예시

- `docs(fsd): add fsd guide and links`
- `chore(react-query): add query client provider`
- `feat(retro): add retrospective list query`
